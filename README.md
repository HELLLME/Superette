# Superette 🛒  
*A modern and responsive showcase website for supermarkets.*

## 🚀 Overview
Superette is a web-based showcase platform designed for supermarkets looking to establish a strong digital presence. It features a clean user interface, product displays, and essential business information to provide a seamless and engaging experience for customers.

## ✨ Features
✅ User Authentication (login form with email and password val****idation)
✅ Product Management (view list of products, add new product, view single product details)
✅ UI Components (responsive layout with header, footer, landing page)
✅ Product table with sorting and pagination
✅ Ads slider component
✅ Error page for unknown routes
✅ Styled with Bootstrap and custom CSS
✅ Routing between landing page, authentication, products list, add product, and error page
🔜 User Registration (sign-up and password reset)
🔜 Real Backend Integration (database or API)
🔜 User Roles or Permissions (admin/user distinction)
🔜 Product Images or Rich Details
🔜 Search or Filter for Products
🔜 Profile or Account Management
🔜 Unit or E2E Tests for business logic

## 🛠️ Technologies Used
- HTML, CSS, TypeScript
- Angular Framework

## 📂 Installation & Setup

**Prerequisites:**
- [Node.js](https://nodejs.org/) (Recommended: LTS version)
- [Angular CLI](https://angular.io/cli):  
  Install globally if you haven't already:
  ```bash
  npm install -g @angular/cli
  ```

**Clone the Repository:**
```bash
git clone https://github.com/your-username/superette.git
cd superette
```

**Install Dependencies:**
```bash
npm install
```

**Run the Development Server:**
```bash
ng serve
```
By default, the application will run at [http://localhost:4200/](http://localhost:4200/) and hot-reload on changes.

**Build for Production:**
```bash
ng build --prod
```
The build artifacts will be stored in the `dist/` directory.
